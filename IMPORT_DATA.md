// subsidy_data.csvをスプレッドシートにインポートする手順

1. スプレッドシートを開く
   https://docs.google.com/spreadsheets/d/15kskYhjvdELGRoqtWbf75cuxNdhHDMytysRX98mpVpE/edit

2. 「シート2」を選択

3. ファイル → インポート → アップロード → subsidy_data.csvを選択

4. インポート設定:
   - インポート場所: 「現在のシートを置き換える」
   - 区切り文字: カンマ
   - テキストを数値に変換: はい

5. 「データをインポート」をクリック

これで20件の補助金データが追加されます。
